# マイトレードダッシュボード

自分専用の株価チェック＆AIアドバイスアプリです。プログラミングの知識は不要です。

## できること

- 世界の主要市場（米国・日本・中国・欧州など）の値動きサマリー表示（実際の株価指数を使用、通貨換算の混乱がないよう指数ポイントで表示）
- 米国株はFinnhubの無料WebSocketにより価格がリアルタイムで更新されます（日本株は15秒ごとの自動更新）
- 個別銘柄のローソク足チャート表示（1日／3か月／6か月）※米国株・日本株どちらも対応
- RSI（テクニカル指標）の表示
- 保有銘柄の登録・含み損益の確認
- 保有銘柄について「利確/損切りを検討すべきか」をAIが材料整理してアドバイス
- 最新ニュースから「今日の注目テーマ」と関連銘柄をAIが提案

すべて無料のAPIキーだけで動作します（月額費用はかかりません）。

## 使い方（3ステップ）

### ステップ1: 無料APIキーを2つ取得する

**① Finnhub（米国株の株価・ニュース用）**
1. https://finnhub.io/register にアクセスして無料登録
2. ログイン後のダッシュボード画面に表示される「API Key」をコピー

**② Google AI Studio（AIアドバイス用・Geminiのキー）**
1. https://aistudio.google.com/apikey にアクセス（Googleアカウントでログイン）
2. 「Create API key」ボタンを押してキーを作成し、コピー

どちらもクレジットカード登録不要で、無料枠の範囲で個人利用には十分です。

### ステップ2: アプリを起動する

このフォルダの中の **`start.command`** をダブルクリックしてください。ターミナルの黒いウィンドウが開き、自動的にブラウザでアプリが立ち上がります。

> **初回だけ出るお知らせについて**
> 「開発元を確認できないため開けません」という警告が出た場合は、`start.command` を **右クリック（またはControlキーを押しながらクリック）→「開く」** を選び、表示されるダイアログで再度「開く」を押してください。これはMacの標準的なセキュリティ確認で、一度許可すれば次回からは普通にダブルクリックで開けます。

初回起動時に設定画面が出るので、ステップ1で取得した2つのキーを貼り付けて「保存してはじめる」を押してください。キーはこのパソコンのブラウザ内にのみ保存され、どこにも送信されません。

このサーバーを経由して開くことで、①ブラウザのセキュリティ制限を回避でき、②日本株のデータも取得できるようになります（`index.html` を直接ダブルクリックする方法では日本株が使えません）。

使い終わったら、ターミナルのウィンドウで `Control + C` を押すか、そのままウィンドウを閉じればサーバーは停止します。次に使うときはまた `start.command` をダブルクリックしてください。

### ステップ3: 使ってみる

- 上部の検索欄に銘柄コードを入力してEnter
  - 米国株の例: `AAPL`, `TSLA`, `NVDA`
  - 日本株の例: `7203`（トヨタ）, `9984`（ソフトバンクG）など証券コード4桁
- 「保有銘柄」欄に銘柄コード・株数・取得単価を入力して追加
- 「売却アドバイスを見る（AI）」ボタンでAIによる材料整理を表示
- 「最新ニュースからテーマを提案してもらう」ボタンで注目テーマと関連銘柄を提案

## チャート表示について（重要）

- Finnhubの無料プランではチャート用の過去データが提供されないため、チャート（ローソク足・RSI）は米国株・日本株ともに、このアプリ内のローカルサーバー（`server.py`）がYahoo!ファイナンスのデータを取得して表示しています。
- そのため、**チャートは`start.command`でサーバーを起動しているときだけ表示できます**。現在値や含み損益（米国株はFinnhub、日本株はYahoo!ファイナンス経由）はこれまで通り取得できます。
- 日本株の**個別銘柄ニュース**には対応していません（売却アドバイスはテクニカル指標のみを材料にします）。
- 多少の遅延がある場合があります。

## 注意点

- このアプリが表示する内容は**投資助言ではありません**。あくまで情報整理・参考情報であり、最終的な投資判断はご自身の責任で行ってください。
- 自動売買機能は搭載していません（意図的に、実行系の機能は付けていません）。
- 無料APIキーには利用回数の上限があります。短時間に何度もリクエストすると一時的にエラーになることがありますが、時間を置けば復旧します。

## スマホや友達と共有する方法

### 自宅のWi-Fi内でスマホから見る（今すぐ無料でできます）

パソコンとスマホを同じWi-Fiに繋いだ状態で `start.command` を起動すると、ターミナルに
`同じWi-Fi内のスマホからは http://192.168.x.x:8765 を開いてください` のような行が表示されます。
そのURLをスマホのブラウザ（Safari/Chromeなど）に入力すれば、スマホからも同じアプリが使えます。
このとき、スマホでも一度APIキー（Finnhub・Gemini）の入力が必要です（スマホ側のブラウザに保存されます）。

### 外出先や友達にも使ってもらう（クラウドに無料/格安でホスティング）

1. **GitHubに無料アカウントを作る**（すでにお持ちならスキップ）→ https://github.com/signup
2. `trade-dashboard` フォルダの中身を新しいリポジトリとしてGitHubにアップロード
   （GitHubのサイトで「New repository」→ 作成後の画面でファイルをドラッグ＆ドロップでもアップロードできます）
3. **Renderに無料アカウントを作る**（クレジットカード不要）→ https://render.com
4. Renderで「New +」→「Web Service」を選び、先ほどのGitHubリポジトリを連携
5. 設定項目:
   - Start Command: `python3 server.py`
   - Environment Variables に `APP_PASSWORD` を追加し、友達と共有する合言葉を決めて設定（例: `APP_PASSWORD=好きな合言葉`）
6. 「Create Web Service」を押すとデプロイが始まり、数分後に `https://（名前）.onrender.com` のようなURLが発行されます
7. そのURLと合言葉を友達に伝えれば、友達も自分のスマホ/PCからアクセスできます（友達も自分のFinnhub・Gemini APIキーを入力してもらう必要があります）

**料金について**: Renderの無料プランはクレジットカード登録不要で、月750時間まで無料です。15分間アクセスがないとスリープし、次にアクセスしたときに再起動で30〜60秒ほどかかりますが、個人・友達内利用なら十分実用的です。常時スリープなしにしたい場合は月6ドル程度の有料プランに変更できます。

**セキュリティについて**: `APP_PASSWORD` を設定していないと誰でもアクセスできてしまうため、クラウドに公開する場合は必ず設定してください。保有銘柄などのデータはサーバーには保存されず、各自のブラウザ内にのみ残ります。

## ファイル構成

```
trade-dashboard/
├── index.html      … 画面の骨組み
├── style.css       … デザイン
├── app.js          … データ取得・チャート・AIアドバイスのロジック
├── server.py        … ローカルサーバー（画面配信＋日本株データの中継）
├── start.command    … サーバーを起動するための起動ボタン（これをダブルクリック）
└── README.md        … このファイル
```

## 困ったときは

- 数値が「--」のまま表示される → APIキーが正しいか、右上の歯車アイコンから確認してください
- 日本株が表示されない → `start.command` でサーバーを起動しているか確認してください（`index.html` を直接開いた場合は日本株は使えません）
- チャートが表示されない → 銘柄コードが正しいか確認してください（米国株はティッカー、日本株は証券コード4桁）
- AIアドバイスが出ない → Gemini APIキーが正しいか、1日の無料利用回数の上限に達していないか確認してください
- `start.command` をダブルクリックしても何も起きない → 上記の「初回だけ出るお知らせについて」を参照してください。それでも動かない場合は、お使いのMacにPython 3が入っているか確認が必要です（ほとんどのMacには標準で入っています）
